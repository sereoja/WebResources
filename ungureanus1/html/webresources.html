  • Master Design Lite
    Lets you add a Material Design look and feel to your websites. It doesn’t rely on any JavaScript frameworks and aims to optimize for cross-device use, gracefully degrade in older browsers, and offer an experience that is immediately accessible.

  • Zurb Foundation
    Is a highly advanced, enterprise-grade front-end framework that is ideal for developing nimble, responsive websites. This features-rich framework supports GPU acceleration for smooth, lightning-fast animations and Fastclick.js for fast rendering on mobile devices. It runs on the Sass preprocessor and includes the Foundation-developed data interchange attribute, which lets you load lightweight HTML sections for mobile and “heavier” HTML sections for larger screens.
